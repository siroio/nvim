# Neovim プラグイン キーマップ チートシート

## Leader キー: `Space`

---

## Telescope (ファイル検索)
| キー | 機能 |
|------|------|
| `<leader>ff` | ファイル検索 |
| `<leader>fg` | テキスト検索 (Live Grep) |
| `<leader>fb` | バッファ一覧 |
| `<leader>fh` | ヘルプタグ検索 |

---

## Neo-tree (ファイルエクスプローラー)
| キー | 機能 |
|------|------|
| `<leader>e` | ファイルツリーをトグル |

---

## Yazi (ファイルマネージャー)
| キー | 機能 |
|------|------|
| `<leader>-` | Yazi (ファイラー) を開く |

---

## LSP (言語サーバー)
| キー | 機能 |
|------|------|
| `gd` | 定義へジャンプ |
| `gr` | 参照一覧 |
| `K` | ホバー情報 |
| `<leader>rn` | リネーム |
| `<leader>ca` | コードアクション |
| `<leader>f` | フォーマット |

---

## Trouble (診断表示)
| キー | 機能 |
|------|------|
| `<leader>xx` | 診断一覧をトグル |
| `<leader>xq` | Quickfix リストをトグル |

---

## ToggleTerm (ターミナル)
| キー | 機能 |
|------|------|
| `<C-\>` | ターミナルをトグル |
| `<leader>gg` | Lazygit を開く |
| `<leader>tf` | フローティングターミナル |
| `<leader>th` | 水平分割ターミナル |
| `<leader>tv` | 垂直分割ターミナル |

---

## Barbar (タブ/バッファ)
| キー | 機能 |
|------|------|
| `<A-,>` | 前のバッファへ |
| `<A-.>` | 次のバッファへ |
| `<A-c>` | バッファを閉じる |
| `<A-p>` | バッファをピン留め |

---

## Hop (移動)
| キー | 機能 |
|------|------|
| `s` | 単語へジャンプ |
| `S` | パターンへジャンプ |

---

## WinResizer (ウィンドウリサイズ)
| キー | 機能 |
|------|------|
| `<C-e>` | リサイズモード開始 |

リサイズモード中:
- `h/j/k/l` - ウィンドウサイズ変更
- `Enter` または `q` - 確定

---

## Operator Replace
| キー | 機能 |
|------|------|
| `r{motion}` | モーションで選択した範囲を置換 |
